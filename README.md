# Data-Science-Analytics-Internship-DevelopersHub-Corporation
This repository contains all five tasks completed as part of the Data Science &amp; Analytics Internship at DevelopersHub Corporation. Each task covers a core area of data science including data exploration, visualization, classification, and regression modeling using Python.


***Tasks Overview***

Task 1 — Exploring and Visualizing a Simple Dataset
Dataset: Iris Dataset
Objective: Understand how to read, summarize, and visualize a structured dataset using Python.
Approach:

Loaded and inspected the dataset using .shape, .info(), and .describe()
Created scatter plots, histograms, and box plots to analyze feature distributions and relationships

Key Libraries: pandas, matplotlib, seaborn

Task 2 — Credit Risk Prediction
Dataset: Loan Prediction Dataset (Kaggle)
Objective: Predict whether a loan applicant is likely to default on a loan.
Approach:

Handled missing values and performed data cleaning
Visualized key features such as loan amount, income, and education level
Trained a classification model (Logistic Regression / Decision Tree)
Evaluated performance using accuracy score and confusion matrix

Key Libraries: pandas, scikit-learn, seaborn, matplotlib

Task 3 — Customer Churn Prediction (Bank Customers)
Dataset: Churn Modelling Dataset
Objective: Identify customers who are likely to leave the bank.
Approach:

Cleaned and prepared the dataset
Encoded categorical features (Geography, Gender) using Label Encoding and One-Hot Encoding
Trained a supervised classification model
Analyzed feature importance to understand the key drivers of churn

Key Libraries: pandas, scikit-learn, seaborn, matplotlib

Task 4 — Predicting Insurance Claim Amounts
Dataset: Medical Cost Personal Dataset (Kaggle)
Objective: Estimate medical insurance charges based on personal attributes such as age, BMI, smoking status, and region.
Approach:

Performed EDA to explore correlations between features and charges
Visualized the impact of BMI, age, and smoking status on insurance costs
Trained a Linear Regression model to predict claim amounts
Evaluated model performance using MAE and RMSE

Results:
MetricValueMAE(see notebook)RMSE(see notebook)
Key Libraries: pandas, numpy, scikit-learn, seaborn, matplotlib

Task 5 — Personal Loan Acceptance Prediction
Dataset: Bank Marketing Dataset (UCI Machine Learning Repository)
Objective: Predict which customers are likely to accept a personal loan offer.
Approach:

Explored customer demographics including age, job type, and account balance
Created industry-level visualizations (donut chart, violin plot, heatmap-style bar chart)
Trained a Decision Tree classifier with controlled depth to prevent overfitting
Evaluated using accuracy, classification report, confusion matrix, and feature importance

Results:

Key Libraries: pandas, numpy, scikit-learn, seaborn, matplotlib

🚀 How to Run
All notebooks were developed using Google Colab. To run them:

Open Google Colab
Click File → Upload Notebook and select any .ipynb file from this repository
Upload the required dataset when prompted (datasets are loaded via Colab's file upload)
Run all cells using Runtime → Run All


All required libraries (pandas, numpy, matplotlib, seaborn, scikit-learn) come pre-installed in Google Colab — no additional setup needed.
