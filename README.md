## ✅ Tasks Overview

---

### 📌 Task 1 — Exploring and Visualizing a Simple Dataset

| | |
|---|---|
| **Dataset** | Iris Dataset |
| **Objective** | Understand how to read, summarize, and visualize a structured dataset using Python |

**Approach:**
- Loaded and inspected the dataset using `.shape`, `.info()`, and `.describe()`

- Practiced foundational EDA techniques essential before any ML modeling

### 📌 Task 2 — Credit Risk Prediction

| | |
|---|---|
| **Dataset** | Loan Prediction Dataset (Kaggle) |
| **Objective** | Predict whether a loan applicant is likely to default on a loan |

**Approach:**
- Handled missing values and performed thorough data cleaning
- Visualized key features such as loan amount, income, and education level

**Key Libraries:** `pandas` `scikit-learn` `seaborn` `matplotlib`

### 📌 Task 3 — Customer Churn Prediction (Bank Customers)

| | |
|---|---|
| **Dataset** | Churn Modelling Dataset |
| **Objective** | Identify customers who are likely to leave the bank |

**Approach:**
- Cleaned and prepared the dataset for modeling
- Encoded categorical features (Geography, Gender) using Label Encoding and One-Hot Encoding
- Trained a supervised classification model
- Analyzed feature importance to understand the key drivers of customer churn

**Key Libraries:** `pandas` `scikit-learn` `seaborn` `matplotlib`

---

### 📌 Task 4 — Predicting Insurance Claim Amounts

| | |
|---|---|
| **Dataset** | Medical Cost Personal Dataset (Kaggle) |
| **Objective** | Estimate medical insurance charges based on personal attributes |

**Approach:**
- Performed EDA to explore correlations between features and charges
- Visualized the impact of BMI, age, and smoking status on insurance costs
- Trained a **Linear Regression** model to predict claim amounts
- Evaluated model performance using **MAE** and **RMSE**

**Evaluation Metrics:**

| Metric | Description |
|--------|-------------|
| MAE | Mean Absolute Error — average prediction error |
| RMSE | Root Mean Squared Error — penalizes larger errors more heavily |

**Key Libraries:** `pandas` `numpy` `scikit-learn` `seaborn` `matplotlib`

---

### 📌 Task 5 — Personal Loan Acceptance Prediction

| | |
|---|---|
| **Dataset** | Bank Marketing Dataset (UCI Machine Learning Repository) |
| **Objective** | Predict which customers are likely to accept a personal loan offer |

**Approach:**
- Explored customer demographics including age, job type, and account balance
- Created industry-level visualizations: donut chart, violin plot, and heatmap-style bar chart
- Trained a **Decision Tree** classifier with controlled depth to prevent overfitting
- Evaluated using accuracy score, classification report, confusion matrix, and feature importance

**Key Libraries:** `pandas` `numpy` `scikit-learn` `seaborn` `matplotlib`

4. Click **Runtime → Run All**
